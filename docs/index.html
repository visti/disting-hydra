The beast

Hydra is a CV-controlled audio chord generator: one head becomes many. It takes a single input and harmonises it into a chord by audio — writing the signal into a buffer and reading it back through two to four granular pitch-shifters, each a transposed copy of the input. A pitch tracker measures what the input is actually playing, so every voice lands on an exact target note. Feed it a monophonic source and pick a chord; feed it a drone and freeze it into an evolving pad.

Sibling to Chimera. The freeze smear borrows a trick from PaulXStretch.

Track

A monophonic pitch tracker measures the input's fundamental so voices land on real notes, not fixed intervals.

Voices

2–4 granular pitch-shifters. Stack the scale, or spell an explicit chord type.

Freeze

Stop the buffer and the voices sustain forever. Smear decorrelates the grains into a pad.

Chord brain

Each voice also emits 1V/oct pitch and a gate, so Hydra drives external voices with the same chord.

Getting started

  • Patch an audio source into Input L (and R for stereo — it is summed to mono for shifting). A clear, monophonic source (voice, synth line, bass, guitar) tracks best.
  • On the Routing page set Output L/R; leave Mix at 100% to hear only the chord, or lower it to blend the dry input back in.
  • On the Scale page choose a Scale preset (or hand-pick notes) and a Root.
  • On the Chord page set Voices to 3. You should now hear a triad tracking the input.
  • Patch a 1V/oct CV into Pitch CV (Routing page) to transpose the chord root, quantised to the scale.
No specifications

Unlike Chimera, Hydra reserves a fixed buffer and takes no add-time specifications — just add the algorithm and patch.

Tracking & root

Hydra does not assume you play a fixed note. With Track on, an autocorrelation tracker (running on a decimated copy of the input, roughly 60–1000 Hz) measures the input's fundamental. Each voice is then shifted by voiceNote − inputNote, so the chord lands on true pitches however the input moves. The tracked note is shown top-right of the display as In: ….

With Track off, the input is assumed to be playing the Root note, and shifts are computed from there — cheaper and rock-steady for a fixed drone, but it will be out of tune if the source moves.

Pitch CV & Root

Pitch CV is a 1V/oct input that chooses the chord root: Root + round(CV×12), snapped to the nearest active scale note. Root is the reference note (0V) and, with Track off, the assumed input pitch. The tracker sets tuning; Pitch CV and Root set which chord.

The tracker holds its last reading when it can't find a clear fundamental (silence, noise, thick polyphony). It is monophonic — give it one note at a time.

Scale & quantiser

The scale does two jobs: it quantises the Pitch-CV root, and (in Scale-stack voicing) it is the ladder the chord climbs.

  • Scale — a preset: Major, Minor, the modes (Dorian…Locrian), Harmonic/Melodic minor, Major/Minor pentatonic, Whole-tone, Chromatic, or Custom.
  • Note C … Note B — twelve pitch-class toggles. Active only when Scale is Custom (otherwise greyed out and driven by the preset).
  • Root — the reference MIDI note.

Scale pitch classes are C-based; the Root sets where 0V lands, not which classes are active.

Harmony

How the extra voices are built above the root.

Voicing

  • Scale-stack — voices climb the active scale by Step degrees (2 = thirds → triads/7ths). Diatonic: the chord always belongs to the scale, and it re-spells as you change key. Inversion applies here.
  • Chord-type — voices take intervals from an explicit chromatic table: Maj Min Maj7 Dom7 Min7 Sus2 Sus4 Dim Aug 6 add9 5. Real chord qualities regardless of scale. Chord CV selects the type by voltage (~0.5V per chord, added to the Chord type knob).

Voice lead & Bass

  • Voice lead — octave-shifts each voice to sit near its previous note, so chord changes move smoothly instead of leaping. A bonus: smaller shift ratios mean fewer granular artifacts.
  • Bass — drops the lowest voice one or two octaves for a root beneath the chord (it takes over voice 1; total voice count is unchanged).

The chord

  • Voices — 2 to 4 pitch-shifted copies.
  • Step — scale degrees between stacked voices (Scale-stack voicing only).
  • Inversion — raises the lowest 1–3 voices an octave (Scale-stack voicing).
  • Octave — transposes the whole chord ±2 octaves.
  • Spread — pans the voices across the stereo field (equal-power).
  • Detune — a small per-voice cents offset (root stays pure) for chorusing thickness.
  • Glide — portamento on pitch changes. With Track on and a moving input, voices chase continuously.

Triggers & the gate

Each voice has a low-pass gate — a vactrol-style stage whose envelope drives brightness and level together, so struck notes bloom and darken as they fade. Set LPG to On to enable it; with it Off the voices sound continuously.

Attack–Decay (the swell)

A trigger fires an attack-then-decay envelope. Attack at 0 is an instant pluck; wind it up for a classic swell in. Decay sets the tail. LPG depth blends the gated signal against the raw voice.

Two ways to strike

  • Trig — the chord trigger. One pulse strikes all voices at once (subject to Strum, below).
  • Trig 1–4 — a trigger per voice, so a sequencer or drum pattern can arpeggiate the chord.

Both are live at once: any voice responds to the chord trigger or its own. Per-note triggers ignore Strum and fire immediately.

Motion & freeze

Freeze & Smear

Freeze stops writing the buffer, so the voices keep reading captured audio and sustain indefinitely — a pad from a moment. It engages while the Freeze param is on, or the Freeze in gate is held high (tap a footswitch, hold a gate).

A raw freeze loops its short buffer audibly. Smear fixes that: each grain jumps to a fresh random position in the frozen buffer every time it wraps, decorrelating the loop into an evolving cloud — a time-domain take on PaulStretch's phase randomisation. At 0 it is a plain frozen loop; at 100 it is a full-buffer cloud. Smear only acts while frozen.

Strum · Vibrato · Env→Filter

  • Strum / Strum dir — staggers the voices when the chord Trig fires (Up / Down / Random), turning a chord into a strum or arpeggio.
  • Vib depth / Vib rate — a global chorus vibrato, with each voice given a phase offset so it shimmers rather than moving in lockstep.
  • Env→Filter — an input envelope follower opens the master filter with the dynamics of the source.

Grain & tone

  • Grain — pitch-shifter grain size. Shorter is tighter and more articulate; longer is smoother but softer on transients.
  • Grain sync — when the tracker is confident, snaps the grain length to a whole number of input periods. This aligns grain boundaries to the waveform and markedly reduces the granular warble. Falls back to the Grain time when there is no clear pitch.
  • Filter — a state-variable filter on the wet chord: Off, Low-pass, Band-pass, High-pass.
  • Cutoff / Resonance — the filter frequency and emphasis (also the target for Env→Filter).

Outputs

Beyond the main stereo mix, every voice can leave on its own jacks — Hydra is a chord brain as much as an effect.

  • Out 1–4 — individual audio output per voice, taken post-LPG. A voice with an audio out assigned leaves the main mix so you can process it separately; unassigned voices stay panned in the main output and pass the master filter.
  • Pitch 1–4 — a 1V/oct CV of each voice's note, to drive external oscillators with the same chord.
  • Gate 1–4 — a gate that is high while the voice sounds (its LPG envelope is open, or continuously when LPG is Off).
  • CV out oct — shifts the pitch-CV octave reference so external voices tune where you need them.

Pitch/gate outs are written cleanly (replace); individual audio outs are added to their bus — patch them to unused outputs.

The display

PERFORMANCE VIEW — chord root, tracked input note, active voices

R: the current chord root (note + octave). In: the pitch the tracker is hearing (only shown when Track is on). The voice bars show the chord relative to the input; the waveform at the bottom is the live input feeding the buffer.

Field notes

  • Give the tracker one clean note. Monophonic, sustained sources track best. If tuning drifts on noisy material, switch Track off and play the source at the Root.
  • Smaller intervals sound cleaner. Granular shifting warbles on big jumps. Voice lead and modest Octave settings keep ratios near 1× and the chord smooth; Grain sync tightens it further.
  • Instant pad: hold a note, raise Freeze, turn Smear up, and set a long Attack on the LPG for a slow bloom.
  • Diatonic vs. exact: use Scale-stack when you want the chord to follow a key, Chord-type when you want a specific quality (a dom7, a sus) regardless of scale.
  • Drive a rack: patch Pitch 1–4 and Gate 1–4 to external oscillators and envelopes — Hydra becomes a chord sequencer voiced by your own modules, with the audio chord as a bonus layer.
  • Strummed keys: Chord-type + a long-ish Strum + per-note decay turns single triggers into rolled chords.
